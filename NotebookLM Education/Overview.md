# NotebookLM Education — Overview

NotebookLM education is the top of the funnel for all three audience tracks. People cannot want the product if they don't understand the medium. This is not a minor onboarding problem — most of the target audience has never encountered NotebookLM and has no frame of reference for what it does.

The education series has to do two things simultaneously: explain what NotebookLM is, and show why it matters specifically for climate intelligence. A generic NotebookLM explainer is not enough — the demo has to use the climate corpus so the value is concrete, not hypothetical.

---

## Core Messages to Establish

1. **NotebookLM is not a search engine.** It reads the documents you give it and answers questions about them. No hallucination from the open web — only what's in the corpus.
2. **The corpus is the product.** The Climate Web contains 100,000+ vetted sources built over 35 years. That is what you are accessing — not the internet, not a generic AI, but a curated library of climate intelligence.
3. **It behaves like an analyst, not a search result.** You can ask complex, multi-part questions and get synthesized answers with citations. You can push back, refine, and follow threads.
4. **No technical skill required.** If you can type a question, you can use it.

## Demo Strategy

Every webinar should include a live or recorded demo. The demo should:
- Start with a question the audience would actually want answered
- Show the answer being generated in real time (or close to it)
- Show the citations so the sourcing is visible
- Show a follow-up question to demonstrate the conversational capability

Recorded demo clips should be repurposed as LinkedIn posts and email content.
